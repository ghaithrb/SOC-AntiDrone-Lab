# GNS3 Project Description

This document describes the GNS3 virtual network setup for the SOC Anti-Drone laboratory.

## Overview
The lab simulates a Security Operations Center (SOC) responsible for monitoring and defending against network and drone-related attacks. The virtual topology includes:

- Firewalls (pfSense)
- IDS/IPS servers (Snort, Suricata)
- Monitoring and Logging servers
- Attacker simulation nodes
- Guest and Internal zones

## Purpose
- Train ethical hackers and SOC analysts
- Test attack and defense scenarios in a safe, isolated environment
- Document detection and response procedures
